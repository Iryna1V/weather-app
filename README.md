## About Weather App:

The Weather App is a JavaScript-based application that allows users to get weather information for a given location. 
It utilizes a weather API to fetch real-time weather data and provides a user-friendly interface to display the weather information.

## Functionalities:

- Get current weather conditions, including temperature, humidity, wind speed, and description.
 
- The application displays the temperature in degrees and fahrenheit. You can switch degrees to faringheits

- Display weather information for a given location by entering a city name.

- Responsive design for desktop and mobile devices.

## Tech stack:

- JavaScript
-  HTML
-  CSS
